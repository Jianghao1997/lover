# lovers

## 介绍
情侣任务小程序以及后端服务
微信小程序采用：uni-app，
后端服务采用：springboot，图方便，代码分层简单随便，只用mysql数据库，未采用其他缓存等相关，以及JWT相关！！！ :bowtie: 

## 使用说明

1.  lover-server：为后端服务文件，将文件中的sql脚本运行至自己数据库中，然后修改配置文件即可！
2.  打包为jar，至于你想改成war，直接改pom文件就行。打包完成后直接部署即可！
3.  lover-wechat：配置好小程序id等，以及请求地址即可直接运行！
4.  运行至微信开发者工具，上传发布即可！
